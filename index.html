Form structure for accessible date selection using select

An alternative to type="date" or to using input[type="number"].

Notes:

  • The selects are part of a group, so wrap the whole thing in a fieldset to programatically associate them.
  • Add a legend to give the group a name.
  • Give each select a label.
    • Best choice: a label linked using the id of the select and a for on the label.
    • Second best choice: an aria-label on the select.
  • Don't set the widths of the selects: let the content dictate it.
  • If required by design, visually hide the legend using the sr-only utility class.
